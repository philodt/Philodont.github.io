CHIMÈRES

Organisme créé artificiellement à partir de deux tissus appartenant à des génotypes différents.

Intention:

Création d’une installation interactive dans le cadre de la semaine du code créatif au centre Pompidou ayant pour but d’introduire le code informatique auprès d’un jeune public. Le travail réalisé est lié à l’œuvre de Špela Petrič « Ectogenesis » présentée à l’exposition « La fabrique du vivant », entre février et avril 2019. Les créations des visiteurs sont disponibles ICI.
